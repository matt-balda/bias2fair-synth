# Experimental Scenarios: Synthetic Data and Fairness

## 1. Objective
Evaluate the effectiveness of synthetic data generation as a strategy for bias mitigation in supervised machine learning, and assess its impact on the trade-off between predictive performance and model fairness under controlled and explicitly defined scenarios.

## 5. Experimental Scenarios (STRICT DEFINITIONS)

### Scenario S1 — Baseline (No Intervention)
- No bias mitigation
- No synthetic data
- Train on original dataset

---

### Scenario S2 — Bias Mitigation Only
- Apply preprocessing fairness method (Reweighing or DIR)
- No synthetic data
- Train on adjusted dataset

---

### Scenario S3 — Synthetic Data (Full Augmentation)
- No bias mitigation
- Generate synthetic data from training set
- Final training set = real + synthetic (same distribution)
- Synthetic applied to the entire training dataset, and not only in the disadvantaged group

---

### Scenario S4 — Synthetic Data (Minority Group Only)
- No bias mitigation    
- Generate synthetic samples ONLY for disadvantaged group
- Apply as oversampling
- Final training set = real + synthetic (targeted group only)

---

### Scenario S5 - Combined (Bias Mitigation + Synthetic Data)
- Apply bias mitigation on training data using preprocessing fairness method (Reweighing or DIR)
- Then generate synthetic data from the adjusted dataset
- Final training set = adjusted real + synthetic

---
